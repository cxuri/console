
# Project Blueprint

## Overview

This project is a Next.js application with a simple and clean user interface. It uses Tailwind CSS for styling and includes a basic authentication feature.

## Implemented Features

### Initial Setup

*   **Framework:** Next.js with App Router
*   **Styling:** Tailwind CSS

### UI Enhancements

*   **Homepage:**
    *   Updated the title to "Your Next.js App" and the subtitle to "A starting point for your next great idea."
    *   Added a "Get Started" call-to-action button.
    *   Improved the layout and styling for a more modern look and feel.

## Current Plan

### Authentication Feature

*   **Login/Logout:**
    *   Create a login page with a form to sign in.
    *   Create a dashboard page accessible only to authenticated users.
    *   Implement a navigation bar with links to Home, Login, and Dashboard.
    *   Add a user menu to the navigation bar to show the logged-in user's email and a sign-out button.
*   **Dependencies:**
    *   Install `@radix-ui/react-avatar`, `@radix-ui/react-dropdown-menu`, and `class-variance-authority` for UI components.
